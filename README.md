# 카페인 계산기

## 로컬 실행 (Node.js 필요)

```bash
npm install
npm run dev
```

## 배포 (Vercel)

이 리포지토리를 GitHub에 올린 뒤 [vercel.com](https://vercel.com)에서 Import하면
Vite 프로젝트로 자동 감지되어 배포됩니다. (Build Command: `npm run build`, Output: `dist`)

## AdSense 설정

`src/AdBanner.jsx`의 `data-ad-client`, `data-ad-slot` 값을
본인의 실제 AdSense 게시자 ID / 광고 슬롯 ID로 교체하세요.
